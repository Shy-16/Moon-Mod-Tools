module:classes/BotModule

module:classes/BotModule

Encapsulates all of the methods for loading and running bot modules.

Constructor

new (require("classes/BotModule"))(bot, name, optionsopt, customopt)

Parameters:
Name Type Attributes Default Description
bot Bot

Reference to the Bot that owns this module.
name string

Name of the module. Without custom loading options, this must match the directory that contains the module's index.mjs file.
options Object <optional>
 {}

Module-specific configuration options that the module requires in order to run.
custom Object <optional>
 {}

Custom loading options.

Properties
Name Type Attributes Description
directory string <optional>

Directory containing the module's index.mjs file. Used if you want the module to have a name different from its directory.
indexFile string <optional>

Path to the index.mjs file of the module. Used instead of custom.directory if the directory is not in the bot's directory.
